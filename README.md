## Getting Started

### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```

## Customization

Edit `src/App.jsx` to update:
- Your name and bio
- Skills and technologies
- Projects
- Contact information

Edit `src/styles.css` to customize:
- Colors (CSS variables at the top)
- Fonts
- Animations

## Tech Stack

- React 18
- Vite 5
- Lucide React (icons)
- CSS3 (no frameworks)
